# JavaScript Problem Solving Tasks

This repository contains solutions to 10 basic JavaScript problem-solving tasks using only plain JavaScript.

## Rules Followed

- Used only basic JavaScript (no frameworks, no libraries)
- Custom logic written for each problem
- Focused on understanding and explanation rather than shortcuts
- Code written and explained by myself

## Problems Solved

1. Reverse a String
2. Count Vowels in a String
3. Check for Palindrome
4. Find the Maximum Number
5. Remove Duplicates from an Array
6. Sum of All Numbers in an Array
7. Find Even Numbers in an Array
8. Capitalize First Letter of Each Word
9. Find the Factorial of a Number
10. PingPong Challenge
